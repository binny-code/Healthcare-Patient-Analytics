# Healthcare-Patient-Analytics
This project analyzes patient admissions, billing patterns, hospital workload, and treatment effectiveness using structured SQL queries. It demonstrates real-world healthcare analytics using joins, CTEs, window functions, and time-based analysis.(The schema and Queries both are in .sql file)

### Dashboard Integration
The analytics layer is exposed via SQL views and connected to Power BI/Tableau for real-time visualization. 
Dashboards provide insights into hospital workload, patient stay duration, billing efficiency, and treatment outcomes.
